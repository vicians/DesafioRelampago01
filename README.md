# Portfólio Pessoal — Veronezi

## Visão Geral

Este projeto consiste em um **site de portfólio pessoal** desenvolvido com foco em identidade visual forte, atmosfera técnica e apresentação clara de competências, projetos e posicionamento profissional. O conceito do layout remete a interfaces de sistemas, terminais e arquiteturas digitais, reforçando a ideia de engenharia, estrutura e controle.

O site é totalmente estático, construído apenas com **HTML5 e CSS3**, sem uso de frameworks ou bibliotecas externas, priorizando domínio dos fundamentos e controle total sobre o comportamento visual.

---

## Objetivo do Projeto

O principal objetivo deste portfólio é servir como:

- Apresentação profissional e técnica
- Vitrine de projetos front-end
- Exercício de design de interfaces e identidade visual
- Base evolutiva para inclusão futura de JavaScript, animações avançadas e backend

Além disso, o projeto busca demonstrar cuidado estético, consistência de tema e entendimento de layout responsivo moderno.

---

## Tecnologias Utilizadas

- **HTML5** para a estrutura semântica das páginas
- **CSS3** para estilização avançada, animações, efeitos visuais e responsividade
- **GitHub Pages** para hospedagem dos projetos listados

---

## Identidade Visual e Conceito

O design segue uma estética **dark / cyber / industrial**, utilizando:

- Paleta de cores baseada em preto profundo e vermelho intenso
- Gradientes suaves e fundos radiais para profundidade
- Tipografia em caixa alta para reforçar rigidez e presença
- Elementos com aparência de HUD e interfaces técnicas

O uso de sombras, bordas sutis e efeitos de hover cria sensação de materialidade e resposta ao usuário.

---

## Seções do Site

### Navegação

Navbar fixa com efeito de transparência e desfoque (`backdrop-filter`), garantindo acesso rápido às seções principais sem perder o contexto visual.

### Hero Section

Área de impacto inicial com:

- Título forte e direto
- Subtítulo conceitual
- Frase de posicionamento
- Chamadas para ação claras

O fundo utiliza gradiente vertical e uma malha geométrica sutil para reforçar o clima técnico.

### Arsenal Técnico

Apresentação das competências organizadas por categoria:

- Linguagens e fundamentos
- Ferramentas e sistemas
- Design e experiência do usuário

As habilidades são exibidas como **tags interativas**, com efeitos de hover que simulam ativação ou destaque de módulo.

### Projetos Selecionados

Grid responsivo de cards que exibem projetos publicados no GitHub Pages. Cada card utiliza:

- Imagem de fundo representando o projeto
- Máscara escura com transição
- Tags tecnológicas associadas

O hover revela mais do visual do projeto, criando contraste entre repouso e interação.

### Sobre o Desenvolvedor

Seção textual focada em visão pessoal e abordagem profissional, indo além da parte técnica e contextualizando interesses, valores e forma de pensar sistemas.

### Contato

Área final com chamada direta para comunicação, mantendo a linguagem conceitual do restante do site e reforçando disponibilidade para novos desafios.
